Notes
  1. Don't use ALL CAPS.
  2. Don't abbreviate.
  3. Fields marked with * are required.
  4. Optional fields can be blank.

If you really want to abbreviate, write the full name along with the abbreviation enclosed in parenthesis: Eg. Guimaras State College (GSC)

"N/A"s are automatically added to any empty fields in the downloaded PDS.
